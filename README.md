# DASH (**D**ylan and **A**ndrew's **SH**ell)
This project is an implementation of a shell in C for Systems. 

## Getting Started

Launch Codes

* Clone the Repository
```
$ git clone https://github.com/Andrew1J/DASH.git
```
* Change Directories 
```
$ cd DASH
```
* Run the App
```
$ make
```
```
$ make run
```
